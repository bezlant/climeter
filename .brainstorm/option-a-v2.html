Features

Multi-account

Manage multiple Claude subscriptions. Switch between them from the menu bar.

Auto-switch

Hits 95%? Automatically rotates to your next account. No interruptions.

Zero config

Reads credentials from Claude Code's Keychain. Run /login, done.

Private

No analytics, no telemetry. Only talks to api.anthropic.com. Open source.
